MidiShare Internet Drivers

version française

The MidiShare Internet Drivers provides real-time communication over Internet to MidiShare client applications.
These drivers are two separate shared libraries :

  • msLANDriver.dylib : provides low latency real-time communication restricted to local array networks. It includes automatic recognition of remote hosts.
  • msWANDriver.dylib : provides real-time communication over Internet. Remote host connection is controlled by the user using an .ini file.

Both drivers have the following common characteristics:

  • remote hosts appears as MidiShare slots. To setup connections to these slots, uses the 'msDrivers' application.
  • as remote clients provides a unique identifier with their IP address, their connections state is saved and automatically restored using this identifier.
  • their behavior is controlled by a .ini file (msLANDriver.ini and msWANDriver.ini). These files should be located in your $HOME/MidiShare folder. When not found, default values are used for the parameters (see each driver page).

The MidiShare WAN driver is the base of the Radio Hard Disk project which notably consists in a permanent music broadcasting over Internet.

The MidiShare Internet Drivers are part of the MidiShare project. They have been developed by the Grame's Computer Music Research Laboratory, in collaboration with the company Mil Productions.
